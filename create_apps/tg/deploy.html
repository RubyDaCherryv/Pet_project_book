Запуск бота на сервере

Запуск на сервере tg bot

В данной статье будут использован сервер ruVDS.

ОС Ubuntu 20.04.

Для подключения к серверу будет использован ssh, в терминале прописываем 

ssh <имя_пользователя>@<ip_server>.

Вводим пароль и переходим к первоначальной настройке сервера.

На сервере прописываем команду:

sudo apt update apt install -y htop git build-essential libssl-dev libffi-dev python3-pip python3-dev python3-setuptools python3-venv

Клонируем репозиторий, настравиваем виртуальное окружение.

Переходим в папку с проектом

cd <path_to_bot>

Создаем venv

python3 -m venv .venv

Переходим в venv

source .venv/bin/acriveate

Устанавливаем пакеты

pip3 install -r <req.txt>

Проверяем работу бота из venv

python3 bot.py

Выходим и venv

deactivate
